# 💰 Projeto Financeiro

> **Trabalho Acadêmico:** Este projeto foi desenvolvido exclusivamente como trabalho de faculdade.

Sistema financeiro simples dividido entre uma **API em Python (Backend)** e uma **interface em React (Frontend)**, com ambiente totalmente containerizado.

---

## 🚀 Tecnologias

- **Backend:** Python
- **Frontend:** React + Vite
- **Infra:** Docker & Docker Compose

---

## ⚙️ Como executar

Com o Docker instalado na sua máquina, basta clonar o repositório e executar o comando abaixo na raiz do projeto:

```bash
docker-compose up --build
```

Após os containers serem iniciados, a aplicação estará disponível em:

```text
http://localhost:5173
```
